The Splunk Add-on for Microsoft PowerShell allows a Splunk Enterprise administrator to collect inventory, performance, and alert information from Windows hosts using the Microsoft PowerShell engine as a modular input. You can then directly analyze the data or use it as a contextual data feed to correlate with other data in Splunk Enterprise.
